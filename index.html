Creer un nouveau Fichier

Ouvrir Un Fichier

Pour Ouvrir un fichier sur l'application il vous suffit de cliquer sur la barre de menu, plus précisement sur le fichier et selectionner ouvrir un fichier a la suite de cette action vous une petite fenetre centré sur votre ecran s'afficheras a partir de cette fenetre vous serex en mesure de choisir le chemin ou se trouve le fichier java que vous voulez ouvrir sur l'application, pour vous faciliter la tache nous avon creer un raccourcis sur le clavier pour faire cette operation il vous suffis de tapper la combinaison CTRL+O

NB : L'application permet seulement l Ouverture des fichier java

Enregistrer un fichier

L'enregistrement d'un fichier indique que nous avons creer un nouveau fichier non sauvegardé sur l'application de ce fait pour pouvoir continuer a explorer les fonctionnalité de l'application il nous faut enregistrer ce fichier
Cette operation s'effectue en cliquant sur la barre de menu plus précisement sur le fichier et selectionner Enregistrer Fichier Sous , a la suite de cette action vous verrez une petite fenetre centré sur l'ecran sur laquelle vous pouvez choisir le lieu ou vous voulez enregistrer votre fichier, pour vous faciliter la tache nous avon creer un raccourcis sur le clavier pour faire cette operation il vous suffis de tapper la combinaison CTRL+E

NB : L'application permet seulement L'enregistrement des fichier Java

Sauvegarder un fichier

La sauvegarde d'un fichier indique que nous avons deja Enregistrer notre fichier java mais nous desirons la modifier pour faire quelques chose bien precis
Cette operation s'effectue en cliquent sur la barre de menu plus precisement sur le fichier et selectionner Sauvegarder, pour vous faciliter la tache nous avon creer un raccourcis sur le clavier pour faire cette operation il vous suffis de tapper la combinaison CTRL+S

Copy Paste Cut

Ces fonctionnalité sont très simple a utiliser sur l'application ,COPIER,COUPER,COLLER il vous suffit de faire plus simple CTR+C pour copier ,CTRL+X pour couper, et CTRL+V pour coller
ou vous pouvez passer par la barre de menu en cliquant sur Edition et selectionner l'option qui vous convient

Fermer un Fichier

Le fait de fermer un fichier est consideré comme la chose la plus simple a faire il vous suffit de cliquer sur la barre de menu plus precisement sur fichier et selectionner Fermer Fichier, pour vous faciliter la tache nous avon creer un raccourcis sur le clavier pour faire cette operation il vous suffis de tapper la combinaison CTRL+W

Compiler un fichier

La compilation d'un fichier est l'une des fonctionnalité avancé de notre application, pour compiler un fichier il faut avoir un fichier java ouvert et enregistrer sur l'application , il vous suffit simplement de cliquer sur la barre de menu plus precisement sur Execution et selectionner Compiler, a la suite de cette action vous remarquerez dans la partie designer comme la console de l'application un message d'erreur si votre application contient des erreurs au cas contraire votre compilation seras effectuer avec succès tout en vous affichant un petit message de ce fait vous pourrez passer a la prochaine etape qui est l'execution de votre fichier, pour vous faciliter la tache nous avon creer un raccourcis sur le clavier pour faire cette operation il vous suffis d'appuiyer sur la toucheF5

Execution d'un fichier Java

L'execution d'un fichier java l'une des fonctionnalité avancé de notre application, pour executer un fichier il faut avoir un fichier java ouvert et enregistrer sur l'application , il vous suffit simplement de cliquer sur la barre de menu plus precisement sur Execution et selectionner Executer, a la suite de cette action vous remarquerez dans la partie designer comme la console de l'application un message d'erreur si votre application contient des erreurs au cas contraire votre Execution seras effectuer avec succes, pour vous faciliter la tache nous avon creer un raccourcis sur le clavier pour faire cette operation il vous suffis de tapper la combinaison CTRL+R

Contribuer au Codage

SMART-(JCE) est une application open source de ce fait quelqu'un qui s'y connait en programmation plus precisement Java peut contribuer au developpement de l'application , de ce fait vous n avez qu'a cliquer sur la barre de menu et selectionner Equipe et Contribuer au codage, pour vous faciliter la tache nous avon creer un raccourcis sur le clavier pour faire cette operation il vous suffis de tapper la combinaison CTRL+MAJ+G
